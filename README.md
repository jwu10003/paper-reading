# Paper Reading

AI 领域前沿论文阅读笔记与源码分析，持续跟踪学术进展，积累技术深度。

## 研究方向

- **大语言模型（LLM）**：模型架构、训练优化、推理加速、长上下文、MoE
- **Agent**：智能体架构、工具调用、多智能体协作、规划与推理
- **代码自动生成**：代码大模型、程序合成、自动化开发、代码评测
- **高性能计算（HPC）**：GPU 内核优化、算子开发、并行计算框架、编译优化
- **AI 编译器**：计算图优化、算子融合、自动调优、DSL 设计、代码生成
- **多模态**：视觉语言模型、图像生成、跨模态对齐
- **强化学习与对齐**：RLHF、DPO、奖励建模、安全对齐
- **检索增强生成（RAG）**：向量检索、知识融合、文档问答
- **模型压缩与部署**：量化、蒸馏、剪枝、推理引擎优化
